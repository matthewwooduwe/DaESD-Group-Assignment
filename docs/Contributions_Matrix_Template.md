## Calculation Method

**Scaling Factor Formula:**
Individual Project Mark = (Individual Contribution % ÷ Highest Contribution %) × Group Mark

**Example:**
If Group Mark = 70% and contributions are:
- Person A: 40% contribution → Scaling Factor = 40/40 = 1.0 → Individual Mark = 70
- Person B: 30% contribution → Scaling Factor = 30/40 = 0.75 → Individual Mark = 52.5

**Final Individual Mark:**
= (Individual Project Mark × 0.80) + (Individual Standup Mark × 0.20)
